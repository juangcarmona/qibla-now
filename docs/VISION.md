# Qibla Now — Vision (v1 Android)

## What It Is

Qibla Now is an offline-first Android application built with .NET MAUI that provides:

- Accurate daily Islamic prayer times (Salah)
- Configurable Adhan alarms
- Precise Qibla direction (Compass, Map, Bearing)
- Minimal, privacy-respecting ads

The app works without internet and without user accounts.

## Who It Serves

Muslims who need:

- Reliable prayer times while traveling
- Qibla direction anywhere
- Fully offline capability
- Strong privacy guarantees

## Core Principles

- Offline-first
- No backend
- No authentication
- No analytics
- No telemetry
- No cloud sync
- No personal data collection
- Location processed only on-device

## Strict Non-Goals (v1)

- Quran content
- Dua collections
- Hijri calendar features beyond prayer time computation
- Social or community features
- Sharing or export features
- Cloud backups
- iOS support
