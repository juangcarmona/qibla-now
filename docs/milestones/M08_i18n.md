# M8 — Internationalization (i18n) + RTL

## Decision Locks

* All user-facing text stored in resource files
* Arabic supported with RTL layout
* No hardcoded UI strings
* Layout must use Start / End alignment

Locked.

---

## Inputs

* docs/PRD.md (Localization section)

---

## Scope

* Move all UI text into resource files
* Add supported languages
* Implement language override setting
* Ensure RTL layout works correctly

Supported languages:

* English
* Arabic
* Spanish (if included in PRD)

---

## Non-Scope

* Dynamic translation
* Network translation services

---

## Implementation Rules

* No literal strings in UI
* UI must bind to localized resources
* Layout must support RTL automatically
* Core project must remain language-neutral

---

## Verification

Manual:

* Switch app language to Arabic
* Validate layout mirroring
* Validate navigation works correctly
* Confirm no clipped or overlapping text
* Confirm all strings are localized

Build must pass.

---

## Definition of Done

* Arabic fully usable
* All strings localized
* No literal strings remain in UI
* RTL layout renders correctly
* Output token: COMPLETE

---

## Files Allowed

* QiblaNow.App Resources
* QiblaNow.App Pages
* QiblaNow.App ViewModels
* QiblaNow.App Settings
