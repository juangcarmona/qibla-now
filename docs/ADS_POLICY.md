# Ads Policy — Qibla Now

Ad provider: Google AdMob

Rules:

- Banner only
- Adaptive banners allowed
- No interstitial
- No rewarded

Privacy:

- No analytics
- No Firebase Analytics
- No location sent to ads
- Non-personalized ads (NPA) only
- UMP consent flow implemented where required

Placement:

- Bottom of Times tab
- Bottom of Compass tab
- Bottom of Map tab

Not allowed:

- Permission screens
- Alarm handling UI
- Settings save flow
- Modal dialogs

Refresh:

- No app-driven refresh loops
- SDK-controlled refresh only
- One banner instance per tab

Ads may show even if location not configured.
