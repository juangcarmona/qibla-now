## Support

Pour les questions de support, les demandes de fonctionnalités ou les signalements de bugs, vous pouvez nous contacter via les canaux suivants.

### Commentaires dans l'application

Utilisez l'option de commentaires dans l'app : **Paramètres → Commentaires**.

### Boutique d'applications

Laissez un avis ou envoyez un message via la fiche de la boutique.

### Questions sur la confidentialité

Pour les questions relatives à la confidentialité, consultez la [Politique de confidentialité](/fr/privacy).

---

Qibla Now est maintenu par un seul développeur. Les délais de réponse peuvent varier.
