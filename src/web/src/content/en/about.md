## About Qibla Now

Qibla Now is an Islamic utility app for Android, built for reliability, privacy, and simplicity.

The app provides accurate prayer times, precise Qibla direction, and local alarms without requiring accounts, tracking, or an internet connection.

## Built for trust

Qibla Now is designed around a simple principle: a religious tool should not collect data, depend on remote servers for core functionality, or require identity to function.

All calculations run on the device. All preferences are stored locally.
