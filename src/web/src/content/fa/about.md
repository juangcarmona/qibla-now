## درباره Qibla Now

Qibla Now یک برنامه ابزار اسلامی برای اندروید است که برای قابلیت اطمینان، حریم خصوصی و سادگی ساخته شده است.

این برنامه اوقات نماز دقیق، جهت قبله با دقت و آلارم‌های محلی بدون نیاز به حساب کاربری، ردیابی یا اتصال اینترنت ارائه می‌دهد.

## ساخته شده برای اعتماد

Qibla Now بر اساس یک اصل ساده طراحی شده است: یک ابزار مذهبی نباید داده جمع‌آوری کند، برای عملکرد اصلی به سرورهای از راه دور وابسته باشد یا برای کارکرد نیاز به هویت داشته باشد.

تمام محاسبات روی دستگاه اجرا می‌شوند. تمام تنظیمات به صورت محلی ذخیره می‌شوند.
