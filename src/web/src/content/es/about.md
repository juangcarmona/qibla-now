## Acerca de Qibla Now

Qibla Now es una aplicación de utilidad islámica para Android, diseñada para la fiabilidad, la privacidad y la simplicidad.

La aplicación proporciona horarios de oración precisos, dirección exacta de la Qibla y alarmas locales sin necesidad de cuentas, seguimiento o conexión a internet.

## Construida para la confianza

Qibla Now está diseñada en torno a un principio simple: una herramienta religiosa no debe recopilar datos, depender de servidores remotos para su funcionalidad principal ni requerir identidad para funcionar.

Todos los cálculos se ejecutan en el dispositivo. Todas las preferencias se almacenan localmente.
