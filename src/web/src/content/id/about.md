## Tentang Qibla Now

Qibla Now adalah aplikasi utilitas Islam untuk Android, dibangun untuk keandalan, privasi, dan kesederhanaan.

Aplikasi ini menyediakan waktu salat yang akurat, arah kiblat yang tepat, dan alarm lokal tanpa memerlukan akun, pelacakan, atau koneksi internet.

## Dibangun untuk kepercayaan

Qibla Now dirancang berdasarkan prinsip sederhana: alat keagamaan tidak boleh mengumpulkan data, bergantung pada server jarak jauh untuk fungsi inti, atau memerlukan identitas untuk beroperasi.

Semua perhitungan berjalan di perangkat. Semua preferensi disimpan secara lokal.
